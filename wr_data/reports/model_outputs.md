# WR Rookie Model — Holdout Predictions (2022–2024)

## Models
- **Bayesian Full (BF):** Bayesian ordinal regression, draft capital + 6 college features
- **Bayesian College (BC):** Bayesian ordinal regression, college features only
- **XGBoost Full (XF):** XGBoost cumulative link, draft capital + 6 college features
- **XGBoost College (XC):** XGBoost cumulative link, college features only

## Features
- draft_capital (sqrt-scaled pick score)
- career_yprr (yards per route run)
- career_targeted_qb_rating
- career_grades_pass_route (PFF)
- career_caught_percent
- career_yards_pg
- career_avoided_tackles_pg
